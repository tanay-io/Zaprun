# Zapier (Boilerplate)

Quick Express + TypeScript boilerplate.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run in development (auto-reloads):

```bash
npm run dev
```

3. Build and run:

```bash
npm run build
npm start
```

Files

- `src/app.ts`: Express app (exported)
- `src/server.ts`: Server entry (starts the app)
